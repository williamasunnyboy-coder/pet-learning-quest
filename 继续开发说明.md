## 一、如何运行

```bash
npm install        # 安装依赖（Node 18+，建议 v22）
npm run dev        # 启动开发服务器 → http://localhost:5173/
```

其他脚本：

| 命令 | 作用 |
|------|------|
| `npm run build` | 生产构建（输出到 `dist/`，已配置子目录 `/pet-garden/`） |
| `npm run preview` | 预览生产构建 |
| `npm run lint` | ESLint 检查（当前 0 问题） |
| `npm test` | 运行 vitest 单元测试（当前 27/27 通过） |

## 二、四端入口（同一套状态，localStorage 同源同步）

| 端 | 地址 | 说明 |

| 🐶 孩子端 | `/` | 养成主界面：完成任务→喂养→进化 |

## 三、目录结构（`src/`）

```
src/
├── App.jsx                 # 路由（AppRouter 按 hash 分发四端）+ 孩子端主壳
├── store.js                # 核心状态 hook(useAppStore) + 宠物定义 + 经验/进化/成就/会员
├── teacher-store.js        # 教师端数据层 + 班级码共享桥（教师↔家长↔学员）
├── teacher-bridge.js       # store ↔ teacher-store 解耦层（main.jsx 注入）
├── state-migrations.js     # localStorage schema 版本迁移
├── pin-crypto.js           # 家长 PIN（Web Crypto SHA-256 + PBKDF2）
├── learn-content.js        # 口算/闪卡/计时学习内容引擎
├── stories.js              # 宠物每日故事数据（独立 chunk）
├── decorations.js          # 装扮道具解锁规则
├── constants/
│   ├── modes.js            # 家庭/校园/家校联合 三种模式常量
│   └── mode-meta.js        # 模式展示元数据（图标/文案/配色）
├── role-prefs.js           # 角色偏好（学生/家长/老师）
├── components/             # 23+ 组件（PetView/ParentView/TeacherView/WatchView/PaywallModal…）
└── __tests__/              # vitest 单测

public/pets/<类型>/lv1~lv8.png   # 宠物美术（已抠透明底，本地托管，6 类×8 阶）
```

## 四、核心机制速览

- **经验 → 进化**：`store.js` 的 `EXP_THRESHOLDS`（7 阈值 = 8 阶段）+ `getPetStage()`。
  经验来源：完成任务、连续打卡奖励、每日登录、周挑战。
- **宠物**：6 类常规（4 免费 + 2 会员）+ 4 节日限定；每类 8 个进化阶段（`lv1~lv8`）。
- **会员付费墙**：`getMembership/setMembership`；稀有宠（`isPremium`）需开通会员，见 `PaywallModal`。
  （演示版「开通会员」直接解锁，真实产品需接支付。）
- **三种模式**：家庭（家长确认）/ 校园（完成即结算）/ 家校联合，见 `constants/modes.js`。
- **图片融合**：宠物图为透明底 PNG，在浅色卡片/深色手表屏均自然融合（已移除 mix-blend-mode）。
